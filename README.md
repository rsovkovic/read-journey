# 📚 Read Journey — Reading Tracker App

**Read Journey** is a modern web application designed for book lovers to track their reading progress, manage their personal library, and visualize their reading habits through interactive statistics.

## 🚀 Features

- **Progress Tracking:** Log your reading sessions and keep track of pages read in real-time.
- **Visual Statistics:** Interactive circular progress bars and charts to visualize your daily and total reading goals.
- **Book Management:** Add new books to your collection, update their status, and organize your library.
- **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop views.
- **Dynamic UI:** Smooth transitions and state management for a seamless user experience.

## 🛠 Tech Stack

- **Frontend:** [Next.js 15](https://nextjs.org/) (App Router), [React 19](https://react.dev/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **State & Forms:** [React Hook Form](https://react-hook-form.com/) with Zod/Hookform Resolvers
- **Components:** [React Circular Progressbar](https://www.npmjs.com/package/react-circular-progressbar)
- **Backend Services:** Firebase (Authentication & Database)

## 📦 Installation & Setup

1. **Clone the repository:**

   ```bash
   git clone (https://github.com/rsovkovic/read-journey.git)


   ```

2. **Install dependencies:**

```bash
npm install
```

3. **Run the development server:**

```bash
npm run dev
```

4.  **Build for production:**

```bash
npm run build
```
